# MICAS
> MinION Classification & Alerting System


## Prerequisites

- NPM & NodeJS.
- Python3 and PIP

## Installation

```sh
$ git clone https://github.com/coadunate/xWIMP.git
$ cd xWIMP
$ cd static
$ npm install
```
## Deployment

- Inside the `xWIMP` directory, you need to go to `server` directory and run the
  following command to deploy the application:

```sh
$ python3 server.py
```
